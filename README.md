# Changelog / Обновления

- Добавлен шаблон `layout`, включающий шапку и боковую панель.  
- Протестирован и исправлен функционал регистрации, логина и логаута.  
  - Исправлена ошибка `sqlalchemy.orm.exc.DetachedInstanceError` при ленивой загрузке атрибута `worker` у пользователя ([подробности](https://sqlalche.me/e/20/bhk3)).  
- Созданы страницы клиента:  
  - Пока без возможности создания заявок.  
  - Динамическая подгрузка существующих заявок.  
  - Для каждой заявки создан отдельный шаблон в `static\templates\client\tickets`, подключаемый через `include` в `index` клиента.  
  - Планируется переработка модели заявки с расширением полей.  
- Произведена частичная очистка файлов стилей и скриптов.  
- В `app.py` добавлен обработчик 404:  
  - Несуществующие страницы перенаправляют на страницу соответствующей роли или на страницу логина.  
- Ограничен доступ к страницам других ролей:  
  - Пользователи не могут заходить на страницы ролей, отличных от их собственной, и перенаправляются на страницу своей роли.
